# 周易实验室 - Uni-app X 前端项目

## 项目简介

这是周易实验室的前端项目，基于 uni-app x 开发，提供传统术数排盘工具。

## 功能模块

### ✅ 已实现
- **大六壬排盘**: 完整的大六壬排盘界面，包含：
  - 四柱信息显示
  - 十二宫位排盘
  - 三传四课展示
  - 神煞标注
  - 关系信息展示
  - 动态数据渲染

- **主题系统**: 完整的主题切换和五行配色系统
  - 🌞 浅色主题（默认）
  - 🌙 深色主题
  - ⭐ 黑金主题
  - 👁️ 护眼主题
  - 🚀 科技主题
  - 📄 简洁主题
  - 🎨 五行配色系统（金木水火土）

### 🚧 即将推出
- 六爻起卦
- 紫微斗数
- 八字排盘

## 项目结构

```
zhouyi_front/
├── pages/
│   ├── index/              # 首页（功能导航）
│   │   └── index.uvue
│   └── daluren/            # 大六壬排盘页面
│       └── daluren.uvue
├── static/                 # 静态资源
│   └── logo.png
├── App.uvue               # 应用配置

```

## 开发指南

### 环境要求
- HBuilderX 4.0+
- uni-app x 运行环境

### 运行项目

1. **使用 HBuilderX**
   - 打开 HBuilderX
   - 导入项目 `front/zhouyi_front`
   - 点击运行到浏览器/模拟器/真机

2. **运行到不同平台**
   - H5: 运行到浏览器
   - App: 运行到 Android/iOS 模拟器或真机
   - 微信小程序: 运行到微信开发者工具

### 页面说明

#### 首页 (`pages/index/index.uvue`)
- 功能模块导航入口
- 卡片式布局展示各个功能模块
- 支持点击跳转到对应功能页面

#### 大六壬排盘页面 (`pages/daluren/daluren.uvue`)
从 `front.html` 迁移而来，主要特性：
- 响应式布局，适配多种屏幕尺寸
- 12 宫位排盘展示
- 三传四课居中显示
- 神煞信息纵向排列
- 关系信息详细标注
- 优化的字体大小，确保可读性

## 样式规范

### 颜色系统
- 主色调: `#212529` (文字)
- 辅助色: `#6c757d` (次要文字)
- 强调色: `#dc3545` (课传地支)
- 功能色:
  - 绿色 `#198754`: 将神
  - 蓝色 `#0d6efd`: 天盘
  - 灰色 `#f8f9fa`: 背景

### 字体大小
- 大标题: 24px
- 中标题: 18px
- 正文: 14px
- 小字: 12px
- 极小字: 10px
