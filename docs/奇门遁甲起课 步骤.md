# 奇门遁甲排盘规则

## 概述

奇门遁甲排盘的核心算法规则，包括：

- **定局算法**：根据节气、日期确定阴阳遁、三元、局数
- **排盘算法**：生成地盘、天盘、人盘、神盘的规则体系

## 核心概念

### 九宫（Palace）

```
西北(6)  北(1)  东北(8)
西(7)    中(5)  东(3)
西南(2)  南(9)  东南(4)
```

## 排盘步骤

### 基本流程

1. **确定阴阳遁**：根据节气判断
2. **确定三元**：根据日地支判断
3. **确定局数**：根据节气和三元查表
4. **排布地盘**：天干按戊己庚辛壬癸丁丙乙顺序排列
5. **确定直符和直使**：根据时地支确定
6. **排布天盘**：天盘天干随直符旋转
7. **排布人盘**：八门根据时辰旋转
8. **排布神盘**：八神根据直符位置排列

### 详细规则

#### 地盘排布
地盘天干固定排列：
- 戊永远在中宫（5宫）
- 己永远在坤宫（2宫）
- 其余七干按顺序：庚辛壬癸丁丙乙分布在乾坎艮震巽离兑七宫

#### 直符直使确定
- **直符**：根据局数确定，如一局直符为戊，二局为己，以此类推
- **直使**：直符所在宫位对应的门为直使
- 时辰轮转：按时地支顺序轮转

#### 天盘旋转规律
- **阳遁**：天干顺时针旋转
- **阴遁**：天干逆时针旋转
- 旋转步数：根据时辰与直符的关系确定

#### 八门飞宫
八门按固定顺序飞宫：休、生、伤、杜、景、死、惊、开
- 从直使门开始，按时辰顺序飞宫
- 阳遁顺飞，阴遁逆飞

#### 八神排布
八神从直符宫开始排列：直符、腾蛇、太阴、六合、白虎、玄武、九地、九天
- 按九宫顺序排列
- 中宫不排神

## 特殊情况处理

### 节气交接处理
- **节气当日**：以节气交接的具体时刻为准
- **早子时**：23:00-24:00 归入当日
- **晚子时**：00:00-01:00 归入次日

### 闰月处理
- **闰月排盘**：按实际月份的节气确定
- **节气不变**：闰月不影响节气的排列
- **三元计算**：以实际日干支为准

## 扩展排盘方法

### 传统变化方法

1. **置闰法**：处理符头与节气的差异
2. **拆补法**：以节气为核心的定局方法
3. **随机起局**：支持用户输入数字起局
4. **茅山派**：以真太阳时为准的排盘法
5. **转盘法**：根据不同时辰转换盘面
